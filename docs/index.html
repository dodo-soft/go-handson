# Go Hands-On @uphy --- ## 前準備 SDKと開発環境を整えます。 <small> ※SDKはインストーラでもインストールできますが、 みんなanyenvが好きだと思うのでgoenv使います。 </small> --- ## goenv `anyenv`インストール ```console $ git clone https://github.com/riywo/anyenv ~/.anyenv $ echo '# anyenv' >> ~/.bashrc $ echo 'export PATH="$HOME/.anyenv/bin:$PATH"' >> ~/.bashrc $ echo 'eval "$(anyenv init - zsh)"' >> ~/.bashrc $ exec $bash -l ``` <small>※↑シェル変えてる人は適宜変更して</small> `goenv`、Go 1.11.1をインストール ```console $ anyenv install goenv $ goenv install 1.11.1 $ goenv global 1.11.1 ``` --- ## Visual Studio Code(任意) 以下２つをインストール 1.
